# 🗂️ 파일 및 디렉토리 생성/삭제/복사/이동

## 📌 파일 생성
```bash
touch file.txt
```

## 📌 파일 삭제
```bash
rm file.txt
```

## 📌 디렉토리 생성
```bash
mkdir myfolder
mkdir -p a/b/c
```

## 📌 디렉토리 삭제
```bash
rmdir emptyfolder
rm -r folder
```

## 📌 파일 복사
```bash
cp file1.txt file2.txt
cp -r dir1/ dir2/
```

## 📌 파일 이동 또는 이름 변경
```bash
mv file.txt newfile.txt
mv file.txt ~/Documents/
```
